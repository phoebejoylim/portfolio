Style Guide

These are some key features that represent me in my portfolio.
Everything in this styleguide is meant to be thoughtful, fun, and unafraid of colour.

Style sections

  1. Interactive Elements
  2. Text Elements

Navigation

CSS selector:

.nav-link

Sample code:

<a href="#" class="nav-link">Homepage</a>

Rendered element:

Homepage

In-text link

CSS selector:

a:link, a:visited

Sample code:

<a href="#">This sick link.</a>

Rendered element:

This is just an ordinary paragraph. Nothin' special. Just move along to This sick link.
